# Profile Management System

A clean and responsive Profile Management Web Application built using Next.js, Tailwind CSS, and a lightweight Node.js + Express backend. This project displays student or professional profiles with a modern UI, visual resume, carousel sliders, and individual profile pages.

---

## Features

### Modern Profile Layout
- Gradient banner with contact details
- Circular profile picture with border ring
- Resume download support
- Tab switcher for Case Studies / Projects
- Fully responsive design

### Visual Resume Section
- Smooth scroll to video section
- Video thumbnail with overlay buttons
- YouTube-style label and controls

### Dynamic Carousel
- Slick slider with custom arrows
- Case Study and Project cards
- Mobile-friendly layout

### Backend API
- Express server for profile data
- Clean routing structure
- Easily extendable for database usage

### Individual Profile Pages
- Each profile has a dedicated page
- Organized folder structure
- Easy to add more profiles
